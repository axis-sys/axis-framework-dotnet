# AxisTypes — rule map (value objects)

An Axis **value object** is a string-backed type generated by the `[ValueObject]` attribute: the generator
emits the conversions, the equality and `TryParse`; the author writes only the private constructor (the
validation), the backing property and — for ids — a `New` factory.

This skill **does not restate** the invariants nor carry code — it **routes**. Each map row points to the
canonical rule (in English) under `rules/framework/0-foundations/axis-types/`; the AI opens **only** the rule
the context requires.
